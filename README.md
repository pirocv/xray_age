# xray_age
- Predict biological age from frontal CXR image.


## supported file type
- jpeg
- png

## usage
- Download pre-trained model
  - https://riken-share.box.com/s/0zddg8a5242cffgpr84x2y5lnoqo93h7

```python
python predict.py path_to_cxr_image
```
## requirements
- Python3 (3.8.5)
- Pytorch (1.4.0)
- Torchvision (0.5.0)
- Fastai (1.0.61)
